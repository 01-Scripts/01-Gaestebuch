h1. 01-Gästebuch

Mit dem "_01-Gästebuch_":http://www.01-scripts.de/01gbook.php ist es möglich schnell und einfach ein eigenes, individuelles Gästebuch einzurichten. Dabei müssen Sie sich nicht von vorgegebenen Eingabefeldern einschränken lassen, sondern können über den Administrationsbereich (01ACP) bequem beliebige Eingabefelder für ihr Gästebuch anlegen.
"Weitere Informationen...":http://www.01-scripts.de/01gbook.php
